Energy management and monitoring equipment for smart grids using SST

The major objective of this work is to develop a compact energy routing device used for residential applications. Designing and development of solid state transformer with suitable topology that can integrate the interface for PV installations, battery storage and vehicle to grid system, this is done by re-purposing SST within current EV charging topology.
The Solid State Transformer (SST) is a new concept that has been on trial in this avenue in the last decade. ... A high frequency transformer assisted by MOSFET or IGBT based bridge circuits lie at the heart of every solid state transformer. The increased frequency of operation reduces size of the transformer considerably.The power switches used in the SST offer and additional degree of control over the power flow thorough the device which can be exploited for Smart grid applications. The increasing energy demand parallel by an increasing global temperature resulting from excessive green house gases, has made renewable energy the front runner in supplying our energy demand. So SST is regarded as most important technology for future smart grids, renewable energy and EV sector.
The energy management and isolation system developed in this project utilises a dual active bridge to provide a platform for control of power flow from/to a house connected to the grid remotely. The transformer is designed as a 1:1 isolation transformer for protection purposes, with the MOSFET bridges creating a 50kHz square pulse AC for the transformer. The system acts like a controllable DC bus the power flow through which is controlled using single phase shift modulation method. The system is planned to act as a controller for all DC power sources and storage in a household, giving the utility the opportunity to control the active and reactive power flow to and from the grid to a house. The block diagram of proposed system is given below.
The major specifications of system are Primary and secondary input DC voltage 400V, isolation transformer turns ratio 1:1, transformer operation frequency 50kHz, system efficency >95%, Primary/secondary side maximum current rating 12.5A, power rating 5kW. Control and navigation of simple robotic system

As an intern at UST Global pvt Ltd developed and simulated an automatic navigation and mapping system for indoor robotic navigation. The main idea of project is to find suitable path finding algorithm for the bot. This is implemented in the ROS (Robotic operating system) platform in the Gazibo simulator using inbuilt turtlebot.
... The first step in any navigation problem is to find the map of locality and then navigate to the destination in shortest path possible. For thi purpose we use both hardware and software components. The hardware part mainly consits of a lidar scanner which will give as the data regarding direction and distance to the nearby obstacle. Based on this data a map is build and to this map a path finding algorithm is applied to find the shortest distance to destination. There are mainly two main approach one in which lidar is initially used to create a map of locality and to this defined locality automatic navigation is carried out. Another approach in which map building and navigation is done simultaneously, this is especially useful for unknown areas and interplanetary exploration. The second method requires regerous optimization and efficent computing to achieve satisfiable performance.
The major path finding algorithm that tried includes dijkstra's algorithm and A* algorithm. A* is found to provide much faster navigation. For the second method explained above we tried the multiple rapidly exploring randomized tree algorithm and found that it requires more time and computation to arrive at result. Further we studied on the impact of dynamic obstacles on these algorithms and tried to improve its dynamic performace with the aid of computer vision.

Simulation of intelligent traffic system for self-driving vehicles

This is a study based on the simulation in SUMo (Simulation of Urban MObility), which is an open source traffic simulation package. The proposed system is suitable for semi and fully autonomous vehicles and make use of vehicle to vehicle as well as vehicle to traffic controller communications. ... Transportation plays a major role in shaping economic development of a region. When the number of vehicles increased there is a need of a system that effectively controls the trac. Thus came the system that uses sign boards and traffic light signals to control the traffic . As the technology advances the need for more efficent system to replace the existing trac control system arises . This idea brought us to the concept of wireless trac control system. There's little doubt that the mass adoption of autonomous or self-driving vehicles will change the automotive and transportation industries forever. But the road to driverless cars is not an easy one and there are many engineering, legal and ethical challenges that must be overcome before we can enjoy the benefits of autonomous vehicles.
So to replace the existing system we are suggesting a wireless controlled traffic system . The two major components of this system is a transmitting module which is placed at specific locations in the road and a reciever module which is fitted in each vehicle. Transmitting module consists of a radio frequency transmitter and a microcontroller which is programmed with an algorithm that generates specific encrypted code and send it via transmitter. The code contains some binary pattern that specifies in which direction the movement is allowed at the time a vehicle is approching the intersection. Receving module is implemented in the vehicle ,it also contains a microcontroller and a receiver system. The reciever system recives the signals and match it with the signal generated by the microcontroller. The vehicle also contains a gps module which will give the direction of movement of vehicle and its distance towards the transmitter. If the code matches the vehicle is allowed to approach the junction at any desired speed. If the code mismatched the microcontroller generate additional signals that controls the speed of vehicle so that it approches the intersection with a delay.
By this system we can effectively control the traffic in intersections. This idea can be expanded to more effective traffic control. For example , in a zebra crossing if a pedestrian is in , the sensors associated with it detects it and give signals to vehichles comming towards it and thus slowing down the vehicle. In another contest it is mandatory for vehicles to slow down near school regions for that sometimes humbs are provided. If we establish a wireless system there is no need for speed brakers at all , thus providing a smooth driving experience .

This is also developed as a hobby project. This is basically a two axis plotter implemented by stepper motors. An image is converted to Gcode using a software tool and then it is then translated into coordinates and given to stepper motor through an arduino microcontroller.
